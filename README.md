## 📝 Про проєкт
**Murra Meridian** — це сучасний, швидкий та адаптивний вебсайт-вітрина для демонстрації колекції ексклюзивних годинників. Проєкт реалізовано за концепцією **SPA (Single Page Application)**, де весь контент завантажується миттєво без перезавантаження сторінки, забезпечуючи бездоганний користувацький досвід (UX).

> **Особливість дизайну:** Інтерфейс виконано у глибоких смарагдово-темних тонах, що підкреслює преміальність та розкіш бренду.

---

## 🔥 Ключовий функціонал

*   ⚙️ **Динамічний рендеринг (10 годинників):** Весь асортимент товарів генерується за допомогою JavaScript з єдиної бази даних (масиву об'єктів). Це дозволяє легко масштабувати магазин та додавати нові позиції за секунди.
*   🎨 **Адаптивні розумні градієнти:** Кожен годинник має свій унікальний фоновий градієнт, який ідеально підлаштовується під кольорову гаму моделі на фотографії (наприклад, глибокий смарагдовий, платиновий, золотисто-бурштиновий чи рожеве золото).
*   💬 **Інтерактивне замовлення в Telegram:** Кнопка покупки автоматично генерує персоналізований шаблон повідомлення (з назвою годинника, його кодом та ціною) та перенаправляє клієнта прямо до чату менеджера.
*   🔄 **Інтеграція з Browser History API:** Реалізовано підтримку стандартних кнопок браузера «Назад» та «Вперед» (`history.pushState`). Переходи всередині SPA коректно записуються в історію, запобігаючи виходу користувача із сайту при спробі повернутися на крок назад.
*   📱 **100% Адаптивність (Mobile First):** Інтерфейс бездоганно відображається на будь-яких пристроях — від ультрабюджетних смартфонів до великих моніторів 4K.

## 🛠 Технологічний стек

*   **HTML5** — семантична розмітка, кастомна SVG-графіка для логотипу.
*   **CSS3** — гнучка сітка (Flexbox & Grid CSS), плавні CSS-анімації переходів, змінні (`:root`) для швидкого керування палітрою.
*   **Vanilla JavaScript (ES6)** — керування станом додатку, обробка подій навігації, динамічна маніпуляція DOM-деревом.
*   **FontAwesome Icons** — преміальні векторні іконки.
*   **Google Fonts** — інтеграція витонченого шрифту `Cinzel` та сучасного `Montserrat`.
